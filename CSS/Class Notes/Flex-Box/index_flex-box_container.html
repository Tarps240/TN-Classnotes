Create a Flex Container

Before the Flexbox Layout module, there were four layout modes:

  • Block, for sections in a webpage
  • Inline, for text
  • Table, for two-dimensional table data
  • Positioned, for explicit position of an element

The flex container properties are:

  1. flex-direction
  2. flex-wrap
  3. flex-flow
  4. justify-content
  5. align-items
  6. align-content

A Flexible Layout must have a parent element with the display property set to flex.

flex-direction

  1. The row value stacks the flex items horizontally (by default from left to right on main axis)
  2. The column value stacks the flex items vertically (from top to bottom on cross axis)
  3. The column-reverse value stacks the flex items vertically (but from bottom to top)
  4. The row-reverse value stacks the flex items horizontally (but from right to left)
  5. The wrap-reverse value specifies that the flexible items will wrap if necessary, in reverse order

Perfect Centering

A flex container with both the justify-content and the align-items properties set to center will align the item(s) in the center (in both axes).
